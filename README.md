# Java入门
### 基本概念
基础核心Java SE，大型开发JavaEE，嵌入式开发Java ME。  
源文件(.java)通过编译器生成二进制的字节码文件(.class)(跨平台的基础)，通过解释器(针对平台)才能执行。  
Java虚拟机：JVM，Java开发工具包：JDK，Java运行环境：JRE  
### java安装
bin目录：常用javac.exe，java.exe  
lib目录：(.jar)库文件  
JAVA_HOME：配置JDK安装路径  
PATH：配置JDK命令文件的位置  
CLASSPATH：配置类库文件的位置
